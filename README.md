# Urpagin.net

A simple, fast, and minimalist personal website.

# Setup

1. Clone the repo
2. Create the `.env` file and add a `WEBHOOK_SECRET=<your_webhook>` field.
3. `docker compose up -d`
4. Up & running at `your_server_ip:42047`
